1. Overview

FLUX is a small, dynamically-typed, interpreted language with Python-style significant-whitespace blocks but a keyword style closer to BASIC — most keywords are written in UPPERCASE (the lexer is case-insensitive, so if, If, and IF all work, but IF is the house style used throughout this document).

Programs are plain text files with a .fx extension, run with:

fx run myprogram.fx

A FLUX program is a sequence of statements. Blocks (the bodies of IF, WHILE, FOR, DEF, CLASS, TRY, EXCEPT, and FINALLY) are introduced with a trailing :, followed by a newline, followed by one indentation level (spaces or tabs — a tab counts as 4 spaces). A block ends when the indentation returns to a shallower level, exactly like Python.

Example
IF 1 < 2:
    PRINT "indented block"
PRINT "back to top level"

2. Literals

2.1 Strings

Strings are delimited by matching single or double quotes and support the usual backslash escapes: \n, \t, \r, \\, \", \'. 

PRINT "Hello, World!"
PRINT 'Line one\nLine two'
FormExampleNotes
Double-quoted"hello"Most common style
Single-quoted'hello'Identical semantics to double-quoted
Escape sequence"a\tb"Tab between a and b

2.2 Integers & Floats

Integers are 64-bit signed whole numbers (i64). Floats are 64-bit IEEE-754 (f64) and require a digit on both sides of the decimal point — 1. or .5 are not valid float literals; use 1.0 and 0.5. 

count = 42
pi    = 3.14159
neg   = -7

2.3 Booleans & None

LiteralMeaning
TRUEBoolean true
FALSEBoolean false
NONEThe absence of a value (prints as NONE)
Truthiness: NONE, FALSE, the empty string "", an empty array [], and an empty dictionary {} are all falsy. Every other value (including 0) is truthy.

2.4 Arrays & Dictionaries

colors = ["red", "green", "blue"]
scores = {"alice": 95, "bob": 87}

PRINT colors[0]        # "red"
PRINT scores["alice"]  # 95

5. Statements & Control Flow

5.1 IF / ELIF / ELSE

score = 72

IF score >= 90:
    PRINT "A"
ELIF score >= 80:
    PRINT "B"
ELIF score >= 70:
    PRINT "C"
ELSE:
    PRINT "F"

5.2 WHILE

i = 0
WHILE i < 5:
    PRINT i
    i = i + 1

5.3 FOR ... IN

FOR iterates over an array. Use the builtin RANGE function to iterate over a sequence of numbers. 

FOR color IN ["red", "green", "blue"]:
    PRINT color

FOR i IN RANGE(5):
    PRINT i        # 0, 1, 2, 3, 4

FOR i IN RANGE(2, 10, 2):
    PRINT i        # 2, 4, 6, 8

5.4 DEF (Functions)

DEF factorial(n):
    IF n <= 1:
        RETURN 1
    RETURN n * factorial(n - 1)

PRINT factorial(5)   # 120

Parameters may have default values:

DEF greet(name, greeting="Hello"):
    RETURN greeting + ", " + name

PRINT greet("Ada")              # "Hello, Ada"
PRINT greet("Ada", "Hi")         # "Hi, Ada"

5.5 CLASS

The constructor method is named INIT. Every method's first parameter conventionally refers to the instance and is accessed inside the method body via SELF. 

CLASS Player:
    DEF INIT(name, level):
        self.name = name
        self.level = level

    DEF describe(self):
        RETURN self.name + " (level " + str(self.level) + ")"

hero = Player("Aria", 10)
PRINT hero.describe()   # "Aria (level 10)"

5.6 TRY / EXCEPT / FINALLY

EXCEPT may be bare (catches anything), name a specific error type to match against the error message, and/or bind the error text to a variable with AS. FINALLY always runs, whether or not an error occurred. 

TRY:
    result = 10 // 0
EXCEPT AS err:
    PRINT "Caught: " + err
FINALLY:
    PRINT "cleanup always runs"
TRY:
    RAISE "custom failure"
EXCEPT Exception AS e:
    PRINT "handled: " + e

5.7 ASK / ANSWER

ASK prints a prompt, blocks for a line of input from stdin, and stores the result in an implicit variable named ANSWER. The input is coerced to an Integer if possible, then a Float, and otherwise left as a String. Each new ASK overwrites ANSWER, so copy it to another variable first if you need to keep it around. 

ASK "What is your name?"
PRINT "Hello, " + ANSWER

6. Errors

FLUX errors happen in one of three stages: lexing, parsing, or execution.

6.1 Lex Errors

ErrorMessageTypical cause
UnexpectedCharacterunexpected character '{ch}' at line {line}, column {column}A character the lexer doesn't recognize, e.g. @, &, or a lone ! not followed by =
UnterminatedStringunterminated string at line {line}, column {column}A string literal missing its closing quote before end of file
InconsistentIndentationinconsistent indentation at line {line}, column {column}A DEDENT that doesn't land back on a previously-used indentation width
InvalidNumberinvalid number literal '{value}' at line {line}, column {column}A malformed numeric literal that can't be parsed as an integer or float

6.2 Parse Errors

ErrorMessageTypical cause
SyntaxErrorparse error at line {line}, column {column}: {message}Any grammar violation — a missing : after a block header, a missing closing )/]/}, an invalid assignment target, or an unexpected token where an expression/statement was expected
Common message text you'll see after SyntaxError: expected ':' after block header, expected newline after ':', expected indented block, expected ')' after arguments, expected statement, expected expression, invalid assignment target.

6.3 Runtime Errors

ErrorMessageTypical cause
UndefinedVariableundefined variable '{name}'Reading a variable that was never assigned
UndefinedPropertyundefined property '{name}'Accessing a field or method that doesn't exist on an object
TypeErrortype error: {message}An operation applied to a value of the wrong type, e.g. indexing a number, or calling something that isn't callable
ArgumentErrorargument error: {message}Wrong number of arguments to a function/native function, or a missing required parameter
IndexErrorindex error: {message}An array or string index that is out of range
KeyErrorkey error: {message}Looking up a dictionary key that doesn't exist
ValueErrorvalue error: {message}A value that's the right type but an invalid value, e.g. dividing by zero, or an invalid literal passed to int()/float()
Exception{message}Raised explicitly via RAISE, or an internal/module-level failure (e.g. a file read error, an unknown module in IMPORT)
All of the runtime errors above (everything except the internal RETURN/BREAK/CONTINUE control-flow signals) can be caught with TRY/EXCEPT. An EXCEPT clause with no type, or typed Exception, catches anything; a named type only matches if that name appears in the error's message text.

7. FXwindows Library

FXwindows is FLUX's standard GUI module. Import it, build a window, add widgets to it, then call .show() to open a real, interactive OS window (built on winit + egui). 

IMPORT fxwindows

win = fxwindows.create_window("My App", 640, 480)
win.add(fxwindows.Label("Welcome!", 20, 20))
win.add(fxwindows.Button("Click Me", 120, 32))
win.show()

FXwindows is built with the gui Cargo feature, which is on by default. Building with --no-default-features replaces .show() with a text-summary placeholder printed to the console — useful on headless machines/CI with no display.
Example: a small settings panel
IMPORT fxwindows

win = fxwindows.create_window("Settings", 400, 300)
win.add(fxwindows.Label("Volume", 20, 20))
win.add(fxwindows.Slider(0, 100, 50, 20, 44))
win.add(fxwindows.Checkbox("Enable notifications", TRUE, 20, 80))
win.add(fxwindows.Button("Save", 100, 32))
win.show()

8.1 Hello, World

FOR i IN RANGE(1, 16):
    IF i % 15 == 0:
        PRINT "FizzBuzz"
    ELIF i % 3 == 0:
        PRINT "Fizz"
    ELIF i % 5 == 0:
        PRINT "Buzz"
    ELSE:
        PRINT i

8.3 Classes, Errors, and Modules together

CLASS BankAccount:
    DEF INIT(owner, balance=0):
        self.owner = owner
        self.balance = balance

    DEF withdraw(self, amount):
        IF amount > self.balance:
            RAISE "insufficient funds"
        self.balance = self.balance - amount
        RETURN self.balance

acct = BankAccount("Ada", 100)

TRY:
    acct.withdraw(250)
EXCEPT AS e:
    PRINT "Transaction failed: " + e
FINALLY:
    PRINT "Balance is now " + str(acct.balance)

IMPORT math
PRINT math.abs(-42)

8.4 Interactive input

ASK "What is your name?"
name = ANSWER

ASK "How old are you?"
age = ANSWER

IF age >= 18:
    PRINT name + ", you are an adult."
ELSE:
    PRINT name + ", you are a minor."
